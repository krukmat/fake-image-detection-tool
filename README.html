üöÄ Quick Start

# Clone the repository
git clone https://github.com/krukmat/fake-image-detection-tool.git
cd fake-image-detection-tool

# Install dependencies
pip install -r requirements.txt
npm install

# Start backend (Terminal 1)
PORT=5001 python app/main.py

# Start frontend (Terminal 2)  
PYTHON_API_URL=http://localhost:5001 node server.js

# Open browser
http://localhost:3000

üîå API Usage

POST /detect

Compare two media files for manipulation detection.

Request: 
{
  "url_original": "https://example.com/original.jpg",
  "url_suspect": "https://example.com/suspect.jpg"
}
Response: 
{
  "manipulated": false,
  "score": 0.98,
  "message": "Images are very similar",
  "media_type": "image",
  "original_dimensions": [800, 600],
  "suspect_dimensions": [800, 600]
}
